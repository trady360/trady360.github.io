How to Get Your Bybit API Key and Secret

  1. Log in to your Bybit account at https://www.bybit.com/.
  2. Hover over your profile icon (top right corner) and click "API" or go directly to API Management.
  3. Click "Create New Key".
  4. Select "API Key Type" as System-generated (for most integrations).
  5. Set an API name, e.g., Trady App.
  6. Under API Permissions, only check:
    • Read-Only âœ…
    Do not check Trade or Withdrawal permissions.
  7. Optionally, bind the IP address (recommended for security). If unsure, leave it empty.
  8. Click "Submit" and complete 2FA verification.
  9. Your API Key and Secret Key will be displayed. Save them in a secure place.
Important: Never share your Secret Key. For security, only enable Read-Only access when using third-party apps like Trady.
