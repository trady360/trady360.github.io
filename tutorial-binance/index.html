How to Get Your Binance API Key and Secret

  1. Log in to your Binance account at https://www.binance.com/.
  2. Hover over your profile icon (top right corner) and click "API Management".
  3. Enter a label for your API key (e.g., Trady App) and click "Create API".
  4. Complete the security verification (email, phone, or 2FA).
  5. Your API Key and Secret Key will be shown. Copy and store them securely — you won’t see the Secret Key again after closing the page.
  6. Under API restrictions, make sure to enable:
    • Enable Reading ✅
    Do not enable trading or withdrawal permissions.
  7. Click Save to apply the changes (you may need to verify again).
Important: Do not share your API keys. Only use them with trusted apps like Trady. For your safety, keep trading and withdrawal permissions disabled.
