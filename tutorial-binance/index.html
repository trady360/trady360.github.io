How to Get Your Binance API Key and Secret

1
Log in to Binance

Access the Binance app and sign in using your credentials.

2
Touch [more Service] button

From the profile Page, select the "More Services" button to explore more options.

3
Search for "API Management"

Use the search bar and type "API" to quickly find the API Management section.

4
Enter a label for your new API key

Enter a label for your new API key, e.g. "Trady", then tap Create.

5
Confirm API creation

Follow the instructions to confirm the API creation via email or security verification.

6
Enter your authentication code

Use your 2FA (Google Authenticator or SMS) to confirm and proceed.

7
Copy your API Secret

Copy the secret key immediately. It will not be shown again.

8
Paste API Key and Secret Key to Trady
Important: Do not share your API keys. Only use them with trusted apps like Trady. For your safety, keep trading and withdrawal permissions disabled.
